CyberSecurity Awareness Chatbot

1. Project Title
CyberSecurity Awareness Chatbot (Part 1)

2. Description
This project is a C# console-based chatbot designed to educate users about basic cybersecurity concepts. The chatbot interacts with users through a text-based interface, providing information on topics such as password safety, phishing, and safe browsing.
The application also includes a voice greeting and visual enhancements like ASCII art and coloured text to improve the overall user experience.

3. System Setup (Beginner Friendly)

Follow these steps to run the project:

  1. Install Visual Studio (Community Edition recommended)
  2. Open Visual Studio
  3. Click "Open a project or solution"
  4. Navigate to the project folder
  5. Open the file:
     CyberSecurityChatbot.sln
  6. Ensure the file `welcome.wav` is inside the project
  7. Click the green Run button or press:
     F5

  The application will now start.

4. Usage Instructions (Test Commands)

Once the chatbot starts, try typing the following:
- how are you
- purpose
- what can i ask
- password
- phishing
- safe browsing

To exit the chatbot, type:
- exit
